# LuxeGifts Website Walkthrough

I have created an attractive and premium frontend for the **LuxeGifts** shop. The website is designed to provide a high-end shopping experience with a focus on visual excellence and smooth interactions.

## 🌟 Key Features

### 1. Modern Design Aesthetic
- **Glassmorphism**: Component cards and the navbar use semi-transparent, blurred backgrounds for a sleek, contemporary feel.
- **Premium Typography**: Uses 'Playfair Display' for elegant headings and 'Outfit' for readable, modern body text.
- **Harmonious Color Palette**: A sophisticated mix of Deep Charcoal, Vibrant Rose Gold (#ff3366), and Soft White.

### 2. Interactive Experience
- **Fluid Animations**: Implemented using `framer-motion` for smooth scroll reveals, floating hero elements, and hover transitions.
- **Dynamic Navbar**: A sticky navigation bar that transforms as you scroll.
- **Responsive Layout**: Fully optimized for desktop, tablet, and mobile screens.

### 3. Product Catalog
The website showcases a wide range of premium gift items:
- **Watches**: Luxury chronographs with detailed descriptions.
- **Wallets**: Handcrafted leather accessories.
- **Gift Boxes**: Elegant presentation sets for special occasions.
- **Stationery**: Minimalist executive writing kits.
- **Umbrellas**: Sturdy, designer wind-masters.

## 🛠️ Technology Stack
- **Framework**: React.js with Vite for blazing-fast performance.
- **Styling**: Vanilla CSS for maximum flexibility and control.
- **Icons**: `lucide-react` for clean, professional iconography.
- **Motion**: `framer-motion` for high-quality animations.

## 📂 Project Structure
- `src/App.jsx`: Main application logic and component assembly.
- `src/index.css`: Comprehensive design system and global styles.
- `public/images/`: High-resolution product imagery generated specifically for this shop.

## 🚀 Getting Started
To view the project:
1. Ensure you have Node.js installed.
2. Run `npm install` (already done).
3. Run `npm run dev` to start the development server.
4. Open your browser at `http://localhost:5173`.
